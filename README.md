## ✨ Features

- **🚀 Instant Calculation**: Calculates percentage increase or decrease between a start and end value.
- **🎨 Dynamic Results**: The result is prominently displayed and color-coded (Green for increase, Red for decrease) for immediate visual feedback.
- **🔢 Raw Difference**: Shows the absolute numeric difference between the two values for additional context.
- **📱 Fully Responsive Design**: Works seamlessly on desktops, tablets, and mobile devices.
- **🌌 Modern UI/UX**: Features a sleek, animated cosmic background and a user-friendly, centered interface.
- **📚 Collapsible Guide**: A comprehensive 3500+ word article on percentage change is available in a "Read More" section, ensuring the UI remains clean and uncluttered.
- **ℹ️ Informational Modals**: Includes easily accessible modals for About, Contact, Usage Guide, Privacy Policy, ToS, and DMCA.
- **📈 SEO Optimized**: Built with SEO best practices, including a full suite of meta tags and JSON-LD schemas for enhanced search engine visibility.

## 💻 Tech Stack

- **Framework**: React 18+ (Functional Components & Hooks)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Core Logic**: Pure TypeScript/JavaScript mathematical functions (no external libraries for calculations).

## 🧮 Core Calculation Logic

The application uses the standard formula for percentage change, ensuring accurate and reliable results:

`Percentage Change = ((New Value - Old Value) / |Old Value|) * 100`

All calculation logic is handled client-side within the browser, ensuring user privacy as no data is sent to a server.

## 🛠️ Setup and Installation

To run this project locally, follow these steps:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/hsinidev/percentage-change-calculator.git
    cd percentage-change-calculator
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Start the development server:**
    ```bash
    npm run dev
    ```

4.  Open your browser and navigate to `http://localhost:5173` (or the port specified in your terminal).
